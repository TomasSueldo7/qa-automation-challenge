# QA Automation Challenge

## Selecciones

- Web: Sauce Demo[](https://www.saucedemo.com/) - Demo e-commerce para E2E testing.
- API: ReqRes[](https://reqres.in/) - API REST pública para user management.

## Tecnologías

- Cypress con Cucumber (Gherkin) para BDD.
- Node.js/JavaScript.

## Cómo correr

- `npx cypress open` para modo interactivo.
- `npx cypress run` para headless.
